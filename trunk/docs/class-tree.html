Class Hierarchy

  • netcdftime.netcdftime.datetime: Phony datetime object which mimics the python datetime object, but allows for dates that don't exist in the proleptic gregorian calendar.
  • netcdftime.netcdftime.utime: Performs conversions of netCDF time coordinate data to/from datetime objects.
  • object: The most base type
    • netCDF4.Dataset: A netCDF Dataset is a collection of dimensions, groups, variables and attributes.
      • netCDF4.Group: Groups define a hierarchical namespace within a netCDF file.
      • MFnetCDF4.MFnetCDF4.Dataset: class for reading a multi-file netCDF dataset.
    • netCDF4.Dimension: A netCDF Dimension is used to describe the coordinates of a Variable.
    • netCDF4.Variable: A netCDF Variable is used to read and write netCDF data.
